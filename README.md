# MVP-assignment, MyFriend

## 애플리케이션 설명

- MVP패턴을 이용한 전화번호 애플리케이션
- 친구, 국가, 태그, 설정탭을 나누어 각 기능들을 수행
- 기업과제를 수행하면서 진행한 프로젝트입니다.

## 기술

- MVP
- Koin
- Retrofit2
- RxKotlin
- Android AC - DataBinding, LiveData, Room, NavigationComponent
